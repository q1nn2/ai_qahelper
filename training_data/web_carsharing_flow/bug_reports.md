# Б1 — Кнопка «Назад» в форме бронирования: обычная стрелка вместо знака «<»

- **ID:** Б1  
- **Приоритет:** Желательный  
- **Статус:** Открыт (по результатам прогона)

## Предусловия

- Перейти на тестовый стенд: https://qa-routes.praktikum-services.ru/

## Шаги воспроизведения

1. Ввести в поле «Откуда»: Хамовнический вал, 18.  
2. Ввести в поле «Куда»: Усачева, 3.  
3. Выбрать режим «Свой».  
4. Выбрать вид транспорта «Каршеринг».  
5. Нажать «Забронировать».

## Ожидаемый результат

Соответствие макету Figma.

## Фактический результат

В Google Chrome и Firefox отображается обычная стрелка вместо знака «<».

## Окружение

- Браузер: Google Chrome v.144.0.7559.133  
- ОС: Windows 11 Pro 21H2, 1920×1080
