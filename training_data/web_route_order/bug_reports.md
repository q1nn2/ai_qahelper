# Баг-репорты

# BUG1 — Не проставляется 0 перед однозначными числами в поле «Часы»

- **ID**: BUG1  
- **Статус**: Открыт  
- **Приоритет**: Желательный  

## Предусловия

- Открыть сервис Яндекс Маршруты: https://qa-routes.praktikum-services.ru/

## Шаги воспроизведения

1. Заполнить поле Часы значением `02`.  
2. Снять фокус с поля.

## Ожидаемый результат

Перед однозначным числом система автоматически проставляет `0`.

## Фактический результат

Система не проставляет `0` перед однозначными числами.

## Окружение

- Браузер: Google Chrome v.144.0.7559.133  
- ОС: Windows 11 Pro 21H2

## Связанные тест-кейсы

- [`T1`](../testcases/TC-hours.md)
- [`T2`](../testcases/TC-hours.md)
- [`T3`](../testcases/TC-hours.md)
- [`T4`](../testcases/TC-hours.md)
- [`T5`](../testcases/TC-hours.md)

# BUG15 — Ошибка в расчётах стоимости поездки на своём авто при расстоянии больше 0

- **ID**: BUG15  
- **Статус**: Открыт  
- **Приоритет**: Критический  

## Предусловия

- Открыть сервис Яндекс Маршруты: https://qa-routes.praktikum-services.ru/

## Шаги воспроизведения

1. Заполнить поле Часы значением `08`.  
2. Заполнить поле Минуты значением `10`.  
3. Заполнить поле Откуда значением `Усачева, 3`.  
4. Заполнить поле Куда значением `Хамовнический Вал, 34`.  
5. Кликнуть на кнопку «Свой».  
6. Кликнуть на иконку «Автомобиль».

Получен расчёт стоимости и времени поездки. **Стоимость = 52 рубля**.

## Фактический результат

Получен расчёт стоимости и времени поездки. **Стоимость = 28 рублей**.

## Окружение

- Браузер: Google Chrome v.144.0.7559.133  
- ОС: Windows 11 Pro 21H2

## Комментарий

Данный баг прослеживается на всех временных интервалах.
