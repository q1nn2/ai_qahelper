### SP1_TK14 / SP1_TK14
- **Название:** Некорректный текст сообщения об ошибке при вводе менее допустимого количества символов в поле "Занятие"
- **Приоритет:** Низкий
- **Статус:** Открыт
- **Предусловия:** 1. Перейти на тестовый стенд: https://code.s3.yandex.net/qa/files/mesto/index.html  
2. Открыть окно редактирования профиля (карандаш)
- **Шаги воспроизведения:**
  1. Кликнуть в строку ввода "Занятие"
  2. Ввести “Х”
- **Ожидаемый результат:** Под полем отображается красный текст об ошибке “Недопустимое количество символов”
- **Фактический результат:** Под полем отображается сообщение:  
"Текст должен быть не короче 2 симв. Длина текста сейчас: 1 символ."
- **Окружение:** - Браузер: Google Chrome v.144.0.7559.133
- ОС: Windows 11 Pro 21H2
- **Комментарий:** Вложения: https://skrinshoter.ru/sZxuACkjqeE

### SP1_TK15 / SP1_TK15
- **Название:** Не отображается сообщение об ошибке при вводе более 40 символов в поле «Имя»
- **Приоритет:** Низкий
- **Статус:** Открыт
- **Предусловия:** 1. Перейти на тестовый стенд: https://code.s3.yandex.net/qa/files/mesto/index.html  
2. Открыть окно редактирования профиля (карандаш)
- **Шаги воспроизведения:**
  1. Кликнуть в поле «Имя»
  2. Вставить строку длиной 41 символ
- **Ожидаемый результат:** Под полем отображается красный текст об ошибке “Недопустимое количество символов”
- **Фактический результат:** При вводе 41 символа система автоматически обрезает значение до 40 символов. Сообщение об ошибке не отображается.
- **Окружение:** - Браузер: Google Chrome v.144.0.7559.133
- ОС: Windows 11 Pro 21H2
- **Комментарий:** Вложения: https://skrinshoter.ru/vZxmQccHvQX

### SP1_TK2 / SP1_TK2
- **Название:** Не сохраняются данные в профиле после редактирования в приложение Mesto
- **Приоритет:** Высокий
- **Статус:** Открыт
- **Предусловия:** 1. Перейти на тестовый стенд: https://code.s3.yandex.net/qa/files/mesto/index.html  
2. Открыть окно редактирования профиля (карандаш)
- **Шаги воспроизведения:**
  1. Кликнуть на строку ввода "Имя"
  2. Ввести “Иван”
  3. Кликнуть на строку ввода "Занятие"
  4. Ввести “Художник”
  5. Нажать кнопку “Сохранить”
- **Ожидаемый результат:** На главной странице в профиле отображается “Иван” и “Художник”
- **Фактический результат:** После нажатия кнопки "Сохранить" отображаются старые данные
- **Окружение:** - Браузер: Google Chrome v.144.0.7559.133
- ОС: Windows 11 Pro 21H2
- **Комментарий:** Вложения: https://skrinshoter.ru/vZxw3zsmBJV
