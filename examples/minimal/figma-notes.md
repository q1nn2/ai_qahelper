# Заметки с макета (Figma MCP / вручную)

Кратко опишите экраны: заголовки, подписи полей, тексты ошибок, состояния кнопок.

Пример:

- Заголовок формы: «Вход».
- Поле email с плейсхолдером «you@example.com».
- Сообщение при пустом email: «Введите email».

Передайте этот файл вторым `--requirement` вместе с `requirements.md`, если не используете `--figma-file-key`.
